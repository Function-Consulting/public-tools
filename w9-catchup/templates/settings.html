Settings

W-9 scan folders

Each folder is searched recursively for PDFs. Add as many as you need (for example K:\W-9s plus any folders on P:\ that hold W-9s). A green check means the folder is reachable right now.

Filename filter (optional)

Only scan PDFs whose filename contains one of these comma-separated words (case-insensitive). Leave blank to scan every PDF. Use this when a scan folder (like P:) holds non-W-9 PDFs you don't want OCR'd — for example w9, w-9, w 9.

Spectrum config file

Path to the CM Accounting Utility config.json (ODBC credentials + operator). Used for vendor matching and the Spectrum writebacks. Leave as-is unless the share moves.

Tesseract OCR (optional)

Path to tesseract.exe, used only when scanning image-only W-9 PDFs (digital PDFs read without it). Defaults to the copy Tallera ships on F:. If missing, scanning falls back to PDF text only.

After changing folders, run an Incremental Scan from the main page to pick up PDFs in the new locations. Removing a folder does not delete already-scanned entries from the cache.
